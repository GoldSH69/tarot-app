⚠️ 아래 순서대로 진행하세요:

  1. 위 해시값을 복사하세요
  2. Cloudflare Dashboard에 접속
  3. Workers & Pages → Worker 선택 → Settings → Variables
  4. ADMIN_HASH 값을 새 해시로 변경
  5. Save 클릭
  6. Worker를 재배포 (Edit Code → Deploy)
