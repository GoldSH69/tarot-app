# 🔮 타로 학습 & 리딩 앱

타로 카드를 공부하고, 다양한 스프레드로 리딩할 수 있는 웹 앱입니다.

## 기능
- 📚 78장 카드 학습 (메이저/마이너 아르카나)
- 🃏 원카드, 쓰리카드, 영타로 리딩
- 📝 리딩 일지 저장
- 📮 전문가 리딩 의뢰
- 🤖 관리자 AI 분석 (Gemini)

## 기술 스택
- Frontend: HTML/CSS/JavaScript (순수)
- Hosting: GitHub Pages
- Backend: Cloudflare Workers
- AI: Google Gemini API
- Storage: localStorage + GitHub Gist

## 배포
1. GitHub Repository에 push
2. Settings → Pages → Deploy from branch
3. Cloudflare Worker 별도 배포

## 안내
본 서비스는 타로 카드 학습과 자기성찰을 위한 도구로 제공됩니다.
